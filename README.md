# Exclusive Bootstrap E-Commerce Website
Responsive e-commerce website built with **HTML, CSS, and Bootstrap**.  
Inspired by a **Figma design**, featuring:

- Product carousel
- Category browsing
- Flash sales section
- Sign-up form

**Note:** This is a design-only project. No JavaScript functionality or backend logic is included.

## Technologies Used:
- HTML5
- CSS3
- Bootstrap 5
- Font Awesome
